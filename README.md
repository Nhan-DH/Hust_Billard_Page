# 🎱 HUST Billiard Page

A simple website built to help **students from Hanoi University of Science and Technology (HUST)** find **reliable billiard places near the university**.

Many HUST students enjoy playing billiards after classes or during free time. However, it is not always easy to know which places are affordable, comfortable, and suitable for students.
This website collects and introduces billiard locations that are popular among HUST students.

## 🎯 Project Purpose

The goal of this project is to:

* Help HUST students discover **trusted billiard clubs**
* Provide basic information about billiard places near the university
* Share recommended locations from student experiences
* Create a simple resource for students who enjoy playing billiards

## 🛠 Technologies Used

This project is built using basic web technologies:

* **HTML5**
* **CSS3**
* Static website (no backend)

## 🚀 How to Run the Website

1. Clone the repository:

```
git clone https://github.com/Nhan-DH/Hust_Billard_Page.git
```

2. Open the project folder.

3. Run the website by opening:

```
index.html
```

in your web browser.
